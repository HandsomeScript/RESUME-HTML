BAB 1 Berkenalan Dengan HTML

Pengertian HTML

HTML Singkatan dari Hypertext Markup Language.

Hypertext itu bermakna text yang lebih dari sekedar text. Karena dapat berfungsi sebagai penghubung ke halaman lain (link).

Language artinya adalah bahasa, tapi bukan "programming language".

Kesimpulannya, HTML itu adalah sebuah bahasa khusus yang ditulis dengan menggunakan tanda-tanda tertentu yang digunakan untuk mempresentasikan halaman website.

Fungsi HTML

HTML berfungsi untuk membuat struktur halaman website.

HTML tidak diperuntukkan mengatur tampilan halaman website.

HTML mutlak harus dikuasai sebelum memahami CSS,Js,PHP,ASP,dan semua yang terkait dengan web design dan web programing

Melihat Sekilas HTML

HTML, CSS, Js itu, Source Code nya bisa bebas terlihat oleh siapapun. Caranya dengan klik kanan pada halaman website dan pilih "View page source".Ketiga jenis kode ini dapat dilihat dalam satu halaman yang sama.

Mengenal Jenis-Jenis Bahasa Pemrograman Web

Ada 5 skill yang harus dikuasai untuk dapat membuat sebuah website yang utuh dan dinamis yaitu, HTML,CSS,JS,PHP,MySQL.

HTML untuk membuat struk halaman website, CSS untuk mengatur tampilan, JS untuk mengatur interaksi, PHP untuk mengelola inputan, MySQL untuk database.

BAB 2 Sejarah Dan Perkembangan HTML

Sejarah HTML

Tim Berners-Lee adalah orang yang pertama kali merancang HTML pada tahun 1989.

Tujuan Tim Berners-Lee merancang HTML adalah untuk kemudahan cara dalam menampilkan hasil penelitian sehingga dapat dilihat oleh orang lain.

Awal Lahirnya HTML

Tim Berners-Lee mempublikasikan dokumen berjudul "HTML Tags" dengan konsep 18 element HTML pada tahun 1991. Beliau merancang HTML dengan menggunakan konsep bahasa markup SGML yang memang sudah ada sebelumnya. SGML singkatan dari "Standard Generelized Markup Language".

SGML merupakan sebuah standard International untuk membuat dokumen dengan tanda (mark) seperti <title>,<p>,<li>,<h1> sampai <h6>.Tag-tag inilah yang digunakan oleh HTML. Tetapi tidak semua fitur SGML digunakan oleh HTML. Yang membedakan antara SGML dan HTML itu adalah adanya atribut "href" pada tag <a>.

Tim Berners-Lee juga Mengembangkan Web Browser pertama yang dinamakan dengan WorldWideWeb dan kemudian berganti nama menjadi Nexus.Web Browser inilah yang digunakan untuk menjalankan HTML.

HTML 1.0

Pada tahun 1993, mulai bermunculan Web Browser seperti Lynx, Mosaic, Arena, dll. Masing-masing web browser ini berbeda-beda dalam mendefinisikan HTML. Hal ini karena belum ada standar baku dalam pendefinisian HTML.

Bersama dengan Dave Raggett, Tim Berners-Lee mengajukan HTML 1.0 kepada badan standarisasi arsitektur Internet International IETF(Internet Engineering Task Force). Tetapi karena melewati batas waktu yang ditentukan oleh IETF, maka HTML 1.0 gagal distandarisasi.

HTML 2.0

Karena draft HTML 1.0 gagal terstandarisasi, maka Lembaga standarisasi IETF membentuk HTMLWG( HTML Working Group ) untuk menyempurnakan draft HTML 1.0. Pada akhirnya HTMLWG berhasil menyempurnakannya dan jadilah HTML 2.0 yang menjadi standar resmi HTML pertama pada tahun 1994.

Pada tahun 1994 ini juga, Tim Berners-Lee membentuk W3C (World Wide Web Consortium) yang tujuannya mempopulerkan penggunaan HTML. Jadi HTMLWG dan W3C itu sama-sama menangani standar HTML.

Perang Browser 1

Singkat cerita pada tahun 1994 itu juga, salahsatu programmer web browser Mosaic yang bernama Marc Andreesen keluar dari NCSA (National Center for Supercomputer Applications) yang mengembangkan web browser Mosaic. Kemudian Marc Andreesen mendirikan perusahaan Mosaic Communications yang kemudian berganti nama jadi Netscape Communications Corp. Selanjutnya, perusahaan ini membuat aplikasi web browser berbayar dan menguasai pasar web browser yang bernama Netscape. Mulai dari sini lah banyak produsen software saling bersaing untuk menjadi yang terbaik.

Dengan kondisi persaingan seperti ini, hingga tiba pada pertengahan tahun 1995, Microsoft merilis sebuah web browser bernama IE 1.0 dan siap merebut pasar web browser. Persaingan ini terlihat jelas saat Netscape merilis fitur JavaScript, disaingi oleh IE dengan fitur JScript. Persaingan ini juga menimbulkan adanya tag-tag HTML non standard yang diterapkan di masing-masing web browser seperti <blink> dan <marquee>.

Dalam kondisi persaingan seperti ini (browser war), tentunya sangat menyulitkan para web programing dalam membuat web. Karena harus membuat kode program lebih dari satu agar bisa berjalan di masing-masing browser itu. Pada akhirnya, HTML 2.0 yang sudah menjadi standard harus direvisi kembali.
