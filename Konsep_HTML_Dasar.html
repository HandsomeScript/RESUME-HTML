BAB 1 Berkenalan Dengan HTML

Pengertian HTML

HTML Singkatan dari Hypertext Markup Language.

Hypertext itu bermakna text yang lebih dari sekedar text. Karena dapat berfungsi sebagai penghubung ke halaman lain (link).

Language artinya adalah bahasa, tapi bukan "programming language".

Kesimpulannya, HTML itu adalah sebuah bahasa khusus yang ditulis dengan menggunakan tanda-tanda tertentu yang digunakan untuk mempresentasikan halaman website.

Fungsi HTML

HTML berfungsi untuk membuat struktur halaman website.

HTML tidak diperuntukkan mengatur tampilan halaman website.

HTML mutlak harus dikuasai sebelum memahami CSS,Js,PHP,ASP,dan semua yang terkait dengan web design dan web programing

Melihat Sekilas HTML

HTML, CSS, Js itu, Source Code nya bisa bebas terlihat oleh siapapun. Caranya dengan klik kanan pada halaman website dan pilih "View page source".Ketiga jenis kode ini dapat dilihat dalam satu halaman yang sama.

Mengenal Jenis-Jenis Bahasa Pemrograman Web

Ada 5 skill yang harus dikuasai untuk dapat membuat sebuah website yang utuh dan dinamis yaitu, HTML,CSS,JS,PHP,MySQL.

HTML untuk membuat struk halaman website, CSS untuk mengatur tampilan, JS untuk mengatur interaksi, PHP untuk mengelola inputan, MySQL untuk database.

BAB 4 Struktur Dasar HTML

Pengertian Tag, Atribut dan Element HTML

Pengertian Tag

Tag itu adalah nama singkat dari Element HTML yang menggunakan kurung sudut buka dan kurung sudut tutup.

Secara teknis, fungsi tag HTML itu untuk menginformasikan kepada web browser tentang penyajian dalam bentuk apa data ditampilkan

Pengertian Atribut

Atribut itu adalah informasi tambahan yang ditempatkan pada tag pembuka dan biasanya berpasangan antara key dan value.

Pada penulisan bagian key nya, bisa menggunakan tanda kutip satu/dua dan juga bisa tanpa tanda kutip. Aturan ini bersifat opsional namun disarankan tetap gunakan tanda kutip.

Element HTML

Istilah element pada HTML adalah merupakan satu kesatuan antara tag, atribut dan konten element.

Konten element sendiri juga bisa berupa element lain. Dalam hal ini disebut sebagai element HTML bersarang

Penjelasan Detail Struktur Dasar Halaman HTML

Perhatikan Kode HTML berikut.

Document Type Declaration (DTD)

DTD ini berfungsi untuk memberikan informasi tentang aturan penulisan dalam HTML.

DTD berasal dari SGML yang merupakan induk bahasa markup.

Penulisan DTD pada tiap halaman HTML sebenarnya opsional saja, tapi karena Web Browser sebagai tempat eksekusinya file HTML itu membutuhkan keterangan type document, maka DTD ini menjadi wajib untuk dituliskan. Jika diabaikan, maka halaman HTML dianggab sebagai halaman web yang tidak sesuai standard atau istilahnya masuk dalam quirk mode. Informasi tentang mode ini akan jelas terlihat jika menggunakan web browser Mozila Firefox.

Sebagai lembaga Pengembang HTML, WHATWG tetap membuatkan DTD pada HTML5 dengan penulisan yang singkat, yaitu <!DOCTYPE html> . Aturan penulisannya juga boleh menggunakan huruf kecil.

Untuk penempatannya harus berada di baris paling atas. Tidak boleh diawali oleh karakter apapun termasuk spasi.

Tag <html>

Tag ini disebut sebagai tag root, karena tag ini menampung semua tag-tag HTML yang ada.

Pada tag root ini juga kita bisa menambahkan atribut "lang", yang akan menjelaskan bahwa isi website kita menggunakan bahasa tertentu. Jika atribut berisi "en" berarti konten website menggunakan bahasa inggris. Jika bahasa indonesia, atributnya kita isi dengan "id". Untuk lengkap daftarnya ada di wikipedia.

Tag <head>

Tag ini berfungsi untuk menampung element-element HTML yang digunakan untuk memberikan informasi tambahan kepada web browser tentang halaman yang sedang ditampilkan, seperti tag <meta>, <title>, dan <link>.
